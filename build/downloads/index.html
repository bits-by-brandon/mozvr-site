Display setup

For optimal performance, it is very important to configure your displays as follows. Failure to do so can create excess judder, latency, and annoyance for the user. We are working on improving all of these aspects and making WebVR truly plug-and-play, but for now, best results require the following manual configuration.

Windows:

  • Display>Screen Resolution
  • Orientation: 90° rotation
  • Multiple displays: Extended desktop
  • Display>Screen Resolution
  • Refresh: 60Hz

Mac:

  • System Preferences>Displays>Display
  • Optimize for: Rift
  • Rotation: 90°
  • Refresh: 60Hz
  • System Preferences>Displays>Arrangement
  • Arrangement: Mirrored
